Point of Sale Project Cheatsheet

Load test data

cd ~/workspace/point-of-sale
mongo test_data.js

Load production data

cd ~/workspace/point-of-sale
mongo prod_data.js

Start credit authorization server

authserver start

Stop credit authorization server

authserver stop

Restart credit authorization server

authserver restart

Build and run unit tests

cd ~/workspace/point-of-sale
mvn test

Build and run integration tests

cd ~/workspace/point-of-sale
mvn failsafe:integration-test

Build and create POS Store executable jar

cd ~/workspace/point-of-sale
mvn package -Pstore

Build and create POS Register executable jar

cd ~/workspace/point-of-sale
mvn package -Pregister

Build and create POS Store and Register executable jars

cd ~/workspace/point-of-sale
mvn package -Pstore,register

Start the POS Store application (background)

cd ~/workspace/point-of-sale
java -jar target/pos-store.jar &

Start the POS Register application (can run multiple instances)

cd ~/workspace/point-of-sale
java -jar target/pos-register.jar register-number
